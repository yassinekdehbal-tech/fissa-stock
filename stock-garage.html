🔧 Configuration Firebase

Pour synchroniser le stock en temps réel entre tous vos appareils, suivez ces étapes :

1. Allez sur console.firebase.google.com
2. Créez un projet (ex: garage-stock)
3. Activez Realtime Database → mode test (30 jours) ou avec règles
4. Dans Paramètres projet → Vos applications → Web → copiez la config
5. Ouvrez le fichier HTML dans un éditeur (Notepad) et remplacez les valeurs VOTRE_... par votre config

const firebaseConfig = {
  apiKey: "VOTRE_API_KEY",
  databaseURL: "https://VOTRE_PROJECT-default-rtdb...",
  projectId: "VOTRE_PROJECT",
  ...
};

Une fois configuré, rechargez la page. Le bandeau orange disparaîtra et le stock sera synchronisé en temps réel. ✅
